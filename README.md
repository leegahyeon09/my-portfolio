# 개인 포트폴리오 웹사이트

HTML, CSS, JavaScript로 제작한 개인 포트폴리오 웹사이트입니다.  
학업 및 프로젝트 경험을 정리하고, 프론트엔드 개발 역량을 보여주기 위해 제작했습니다.

## 사용 기술
- HTML / CSS / JavaScript
- Responsive Web Design
- GitHub Pages

## 주요 기능
- 반응형 레이아웃
- 다크 모드 지원
- 프로젝트 카드 구성
- 연락처 폼(Formspree 연동)

## 포트폴리오 구성
- About: 자기소개 및 개발 방향
- Projects: 진행 프로젝트 정리
- Skills: 핵심 역량 및 경험
- Contact: 연락처 및 메시지 폼
